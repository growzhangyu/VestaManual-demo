Installation
============

This page provides download links for slicing software used in the VESTA workflow. Each tool has its own strengths depending on your needs.

Bambu Studio
------------

Bambu Studio is the official slicing software developed by Bambu Lab. It integrates slicing, printer control, and cloud features into one platform.

**Key features:**

- All-in-one solution (slicing + printer control)
- Built-in printer profiles and material presets
- Seamless integration with Bambu printers
- Beginner-friendly interface

Download:
`Bambu Studio <https://bambulab.com/en/download/studio>`_

---

Ultimaker Cura
--------------

Ultimaker Cura is one of the most popular open-source slicing tools, suitable for a wide range of 3D printers.

**Key features:**

- Very beginner-friendly
- Large user community and support
- Simple and clean interface
- Good default settings

Best for:
Beginners or users who want a quick start with minimal setup.

Download:
`Ultimaker Cura <https://ultimaker.com/software/ultimaker-cura>`_

---

PrusaSlicer
-----------

PrusaSlicer is a powerful and flexible slicing software with more advanced configuration options.

**Key features:**

- Highly customizable parameters
- Advanced control over print settings
- Frequent updates and active development
- Supports multiple printer types

Best for:
Intermediate to advanced users who want more control.

Download:
`PrusaSlicer <https://www.prusa3d.com/page/prusaslicer_424/>`_

---

Which one should I choose?
--------------------------

- If you use a **Bambu printer** → use *Bambu Studio*
- If you are a **beginner** → start with *Cura*
- If you want **fine control and advanced settings** → use *PrusaSlicer*
