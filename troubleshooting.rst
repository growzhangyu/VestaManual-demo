Troubleshooting
===============

This section describes common printing issues, their causes, and recommended solutions.

Print Not Sticking to Build Plate
---------------------------------

**Symptoms**

- Model detaches during the first layers
- Filament does not adhere to the surface
- Edges lift or warp

**Possible Causes**

- Contaminated build plate (oil, dust)
- Bed temperature too low
- Incorrect first layer height
- Improper material settings

**Solutions**

1. Clean the build plate

   - Use isopropyl alcohol or a lint-free cloth
   - Avoid touching the surface after cleaning

2. Increase bed temperature

   - For PLA: 50–60°C recommended

3. Adjust first layer height

   - Ensure slight filament compression
   - Adjust Z-offset if necessary

4. Use default settings

   - Avoid manual tuning during initial prints

**Note**

If adhesion fails in the first layer, stop the print immediately and correct the issue.


Stringing (Unwanted Filament Threads)
------------------------------------

**Symptoms**

- Thin strings between printed parts
- Rough or “hairy” surface appearance

**Possible Causes**

- Nozzle temperature too high
- Retraction settings not optimized
- Moist filament

**Solutions**

1. Reduce nozzle temperature

   - Lower in 5–10°C increments

2. Enable or adjust retraction

   - Use default settings if unsure

3. Dry the filament

   - Store in a dry environment
   - Use a filament dryer if available

**Note**

Stringing typically affects appearance only and may not impact functionality.


Print Stops Midway
------------------

**Symptoms**

- Print pauses unexpectedly
- Printer becomes idle before completion
- Incomplete model

**Possible Causes**

- Filament runout or tangling
- Network interruption
- Software or job error

**Solutions**

1. Check filament supply

   - Ensure sufficient filament
   - Confirm no tangling

2. Resume the print (if supported)

3. Restart the print job

   - Use a stable connection

4. Verify network stability

   - Avoid interruptions during printing

**Note**

For long prints, confirm filament availability before starting.
